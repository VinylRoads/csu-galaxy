Галактика расписаний ЧелГУ 

Интерактивная веб-визуализация расписания занятий, превращающая поиск учебной группы в путешествие по звездным системам.

Особенности проекта

Космический интерфейс: Интерактивная карта на JS и CSS с использованием SVG-графики для отрисовки орбит и звездной пыли.


Умный поиск: Мгновенная навигация по группам с эффектом «гиперпрыжка» (размытие и перемещение камеры) к нужной планете.

Автоматизация данных: В систему встроен скрипт на PowerShell, который сам сканирует папку с документами и распределяет группы по факультетам.

Техническая часть
Фронтенд: Чистый JavaScript и CSS3. Для типографики используются шрифты Manrope и Orbitron.

Обработка данных:

build_schedule_json.ps1 — автоматический сборщик базы данных на основе регулярных выражений.

groups.json и schedule.json — структурированные данные о факультетах и файлах расписания.

Серверная настройка: Предусмотрен web.config для корректной отдачи JSON-файлов веб-сервером.

Как пользоваться автоматизацией
Поместите новые PDF-файлы в папку Расписание.

Запустите файл build_all.bat.

Скрипт сам обновит списки групп в groups.json и schedule.json, после чего изменения сразу отобразятся в «Галактике».
